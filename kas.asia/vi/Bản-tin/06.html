Hướng dẫn cài đặt máy in Xprinter

Máy in Xprinter Khổ K80, cổng USB


Bước 1: Click vào nút bên dưới để tải driver máy in, chọn driver đúng với model máy in mình đang cần cài đặt

Tải Driver tại đây
Bước 2: Sau khi hoàn tất tải file cài đặt, các bạn giải nén file và tiến hành cài đặt driver máy in
Bước 3: Nhấp chuột phải và chọn Run as administrator file mới tải về
Bước 4: Chọn I accept the agreement 🡺 Next 🡺 Tiếp tục nhấn Next
Bước 5: Tích chọn Create a desktop shortcut 🡺 Nhấn Next để tiếp tục
Bước 6: Chọn Install
Bước 7: Chọn Finish
Bước 8: Chọn theo thứ tự các số như hình . Lưu ý : Tất cả các máy in K80 lúc nào cũng phải chọn mục 3 là : XP - 80C


Như vậy là bạn đã cài đặt xong máy in bill Xprinter khổ K80.

Bước 9 : Đổi tên máy in bill

Các bạn vào Control panel 🡺 Vào Device & Printer 🡺 Sau đó nhấp chuột phải vào máy in các bạn mới cài đặt xong thường tên mặc định của máy in sẽ là : XP-80C
